HTML & CSS Quick Reference Guide

HTML Tags

<h1> to <h6> — Headings

Used to define section headings, from most to least important.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h6>Smallest heading</h6>

<p> — Paragraph

Wraps blocks of text into paragraphs.

<p>This is a paragraph of text.</p>

<a> — Anchor (Link)

Creates a hyperlink to another page or location.

<a href="https://example.com">Visit Example</a>

<img> — Image

Embeds an image.

<img src="image.jpg" alt="Description">

<ul> & <li> — Unordered List

Creates a bullet-point list.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol> & <li> — Ordered List

Creates a numbered list.

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

<div> — Container

A generic block-level container for layout or styling.

<div class="box">Content here</div>

<span> — Inline Container

Used to style part of text inline.

<p>This is a <span class="highlight">highlighted</span> word.</p>

<code> & <pre> — Code Display

Use these to show code examples. <pre> preserves spacing.

<pre>
  <code>let x = 5;</code>
</pre>

<footer> — Footer Section

Represents the footer for a page or section.

<footer>Copyright 2025</footer>

CSS Properties

color

p {
  color: #333;
}

background-color

body {
  background-color: #f5f5f5;
}

font-family

body {
  font-family: Arial, sans-serif;
}

margin & padding

section {
  margin: 20px 0;
  padding: 10px;
}

border

div {
  border: 1px solid #ccc;
}

text-align

h1 {
  text-align: center;
}

display

.box {
  display: flex;
}

box-shadow

.card {
  box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}
